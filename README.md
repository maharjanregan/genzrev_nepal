# genzrev_nepal

Research website: **Impact of Technology on Nepal’s Gen Z Movement**.

- Built with Astro + MDX
- Deployed via GitHub Pages (GitHub Actions)

## Local dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Pages

After pushing to `main`, enable: **Repo Settings → Pages → Source: GitHub Actions**.
Site URL: https://maharjanregan.github.io/genzrev_nepal/
