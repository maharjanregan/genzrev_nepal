## What this page is (and isn’t)

This page lists **Discord servers and server-level details that are publicly referenced in reporting** (news, blogs, academic/policy analysis, and other public posts).

- We **do not** and **cannot** pull private Discord chat logs.
- We only summarize what **public sources explicitly report**.
- Completeness is **best-effort**, not “entire web.” If a server isn’t named publicly, it won’t appear here.
